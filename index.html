CONCE PTOS BASICOS DE INGIENERIA DE SISTEMA

INGIENERIA

La ingeniería consiste en aplicar los conocimientos científicos a problemas Reales. Otro concepto es el conjunto de teorías y técnicas que permiten el aprovechamiento práctico del conocimiento científico, la ingeniería integra la ciencia, el técnico y el ingenio aplicando prácticas concretas, orientadas a la solución problemas.

SOFTWARE

El concepto de software hace referencia a todos los elementos del computador que no son tangibles pero que contienen las instrucciones necesarias para el procesamiento de la información y para el funcionamiento adecuado de los dispositivos físicos del ordenador, otro término empleado para referirse a este elemento es el programa de programas.

HARDWARE

Hardware es una palabra de origen inglés con las que se hace referencia a toda parte dura de la informática, es decir , a la maquina real utilizada para el procesamiento electrónico de datos” Hardware es la parte tangible, la cual permite que el sistema de cómputo funciones, esto es todos aquellos elementos físicos que se pueden intercambiar de un espacio a otro, manipular y percibir con el tacto, etc. Como ejemplos tenemos los monitores, las tarjetas, las impresoras, cables, chips, etc. 

TIPOS DE SOFTWARE

Software del sistema

Es aquel que se utiliza para controlar el hardware y otros programas, garantizando un adecuado funcionamiento, por ejemplo: el sistema operático, software de redes, entre otros

Software de ingeniería y ciencias

Son programas de mayor complejidad utilizados en áreas específicaciencia, ingeniería e investigación, cuyos resultados se pueden apreciar en tiempo real, garantizado la automatización de procesos, a través del control ejercido sobre una máquina que construye productos o simula evento

Software aplicaciones web

Son hechas con diferentes propósitos teniendo en cuenta las necesidades informativas de los usuarios finales actualmente estas web App se integran con base de datos para acceder a la información particular de cada consumidor

Software de aplicaciones

Es aquel que realiza tareas específicas de procesamientos de datos para el usuario. Por ejemplo: programas contables, etc.

Software incrustados

Este software le brinda al usuario un experiencia de control sobre las acciones específicas de una maquina o sistema en el cual se encuentra implantado. Este tipo de programas se puede encontrar en diferentes electrodomésticos que utilizan tableros digitales para desempeñar funciones particulares, por ejemplo un horno microonda

Software de inteligencia artificia

Es un software especializado en la solución problemas complejos propios de las áreas de la robótica, las redes neuronales sistemas experto

Perfil de un ingeniero de sis​​tema

Perfil profesional

El Ingeniero de Sistemas estará en capacidad de: Analizar, diseñar e implementar sistemas de información.
  Desarrollar software de alta calidad aplicando las actuales técnicas de la ingeniería de software.
  Crear soluciones teleinformáticas tanto en el sector educativo como en el sector productivo. 
 Crear, organizar, dirigir y administrar empresas en el área de las tecnologías tanto informática y telemática. 
 Evaluar sistemas de información y teleinformáticas ya existentes, proponiendo los cambios pertinentes para su optimización. 
 Desarrollar proyectos de investigación innovadores en el campo de las tecnologías informáticas y de telecomunicaciones..

Perfil ocupacional

El Ingeniero de Sistemas podrá desempeñarse en cualquier organización del sector público o privado incluyendo la venta de servicios, que requiera la creación o uso de tecnología de la información y las comunicaciones, así como de la administración de sus recursos. En todos los casos podrán ser actividades propias de nuestro profesional: 
 Consultor en informática, tendrá la capacidad de orientar y formular planes de desarrollo informático y planes estratégicos de tecnologías de la información (PETI) 
 Investigador en el área de nuevas tecnologías de la información y la comunicación (TIC), en los sectores públicos y privados
  Director en sistemas, tendrá la capacidad de gestionar de manera eficiente los recursos informáticos de una organización. 
 Analista, diseñador y desarrollador de software, con las fortalezas técnicas, metodológicas y tecnológicas para la construcción de software de alta calidad y dentro de estándares internacionales.
 Arquitecto de software aplicado a los procesos de negocios de las empresas.
  Tester de software en la industria informática. 
 Diseñador de redes telemáticas, con fortalezas tecnológicas para su construcción con las normas establecida por la organización internacional estándar de la industria (ISO).
