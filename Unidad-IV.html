CONCEPTO DE HARDWARE

EL PROCESADOR

El procesador, esta también conocido como CPU (unidad central de procesamiento), es el encargado de coordinar todos los procesos o acciones, que se realizan para el procesamiento de datos y para la realización de funciones de cada dispositivo, en el transcurso del tiempo han tenido evoluciones, cada día se construyen mejores y con más rápido procesamiento, esto implica el mejoramiento de las funciones de la computadora

ELEMENTOS DEL PROCESADOR

La Unidad de Control

Es la que permite el flujo de datos al interior del procesador y la trasmisión entre dispositivos, teniendo control de todos los recursos del computador. Cada proceso ejecutado dentro de la unidad de control y también recibe instrucciones necesarias para que sean ejecutados. Estas instrucciones estas basadas en micro códigos que al ser recibidas por el procesador le permite ejecutar acciones más complejas.

La unidad aritmética lógica (ALU)

Esta unidad es la encargada de la ejecución o realizar operaciones matemáticas de carácter aritméticas y lógicas. Esta operaciones son fundamentales para procesar los datos que son almacenados en el computador, los cuales son de carácter numérico, las operaciones aritmética (suma, resta, multiplicación y división), las lógicas están determinada (comparaciones y sus opuestos).

TIPOS DE PROCESADOR

Los procesadores internamente están elaborados con una arquitectura muy particular que los dispone a trabajar de una manera distinta a otro procesador. Es esta arquitectura la que define la localización y conexión de sus partes, la ruta de entrada, salida de electricidad, el encendido y apagado de sus transistores, entre otras funciones. Cada tipo de procesador se construye pensando en ser utilizados en máquinas específicas y con programas propios, es decir, para que solo ellos puedan ejecutarlos, lo que ha constituido.

MEMORIAS

La memoria es una parte muy importante del computador, ya que permite el almacenamiento de datos y la ejecución de tareas temporales (durante el tiempo en que el computador se encuentra en uso) o permanente. De este punto de vista físico, se trata de una serie de tarjetas o chips que se ubican en la board del computador. Pueden encontrarse varios tipos de memoria, sin embargo los dos más importantes son la memoria de acceso aleatorio(RAM, Random Access Memory) y la memoria de solo lectura (ROM, read only Memory )

MEMORIA DE ACCESO ALEATORIO

Esta memoria almacena datos e instrucciones de programas mientras la CPU trabaja con ellos, cuando el usuario ejecuta algún proceso, este se carga en la RAM a medida que ese programa vaya necesitando o generando datos, estos se cargan en la RAM, de modo que el programa pueda acceder rápidamente a ellos. La RAM tiene la particularidad de ser voluble, solamente almacena información mientras el computador este encendido. Si hay falla de fluido eléctrico, o si el computador se apaga por cualquier motivo los datos que se encuentran en la RAM y no se hayan almacenado en el disco duro se perderán. Generalmente, la capacidad de la memoria RAM influye de manera directa en la velocidad y rendimiento de una computadora. Así, mientras más memoria RAM tenga el computador va hacer mejor su rendimiento. La capacidad se mide en bytes. Ya sea kilobytes, megabytes, gigabytes, terabytes, entre otros. Byte: cantidad de memoria que se ocupa para almacenar un solo carácter, por ejemplo, una letra del alfabeto o un número. (Norton, 2006)
