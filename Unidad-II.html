CONCEPTO Y TIPOS DE SISTEMAS

Conjunto de elementos o componentes que se relacionan entre sí, para lograr un objetivo o cumplir una función, podemos interpretar que el termino sistemas es muy global, cualquier relación que exista entre algunos elementos de la naturaleza se considera un sistemas. Los sistemas también pueden estar conformados por otros sistemas que son llamados subsistema del sistema principal. Un sistema es considerado un conjunto de partes que se encuentran organizadas, relacionadas que interactúan entre ellos para cumplir un objetivo. Los sistemas tienen elementos de entradas que son procesado para proveer un salida, el resultado se determina de acuerdo al contexto del sistema

TIPOS DE SISTEMAS

sistema físico

Son sistemas mecánicos compuestos por maquinas o hardware , objetos y cosas físicas.

sistemas abstracto

Estan compuestos por conceptos, hipótesis, tesis e ideas, por lo general viven en el pensamiento de la persona, los software son considerados como sistemas abstractos

sistemas cerrado

Estos sistemas no se relacionan con otros sistemas que lo rodea, están blindados a cualquier interferencia del medio en que están. En rigor, no existen sistemas cerrados. Se da el nombre de sistema cerrado a aquellos sistemas cuyo comportamiento es determinista y programado y que opera con muy pequeño intercambio de energía y materia con el ambiente. Se aplica el término a los sistemas completamente estructurados, donde los elementos y relaciones se combinan de una manera peculiar y rígida produciendo una salida invariable, como las máquinas

sistemas abierto

Estos sistemas presentan intercambio con el abiente o con otros sistemas, intercambiando elementos de entrada y de salida. Su conformación está organizada para adaptarse a los cambios que tienen otros sistemas  Sistemas aisl

Elementos y principios de sistemas

Los sistemas es una jerarquía estandarizada la cual siempre estará representada de la misma forma por sus elementos los cuales son los elementos de entradas, los procesos, los elementos de salida, todo sistema tiene esta estructura o su concesión

Elementos de entrada

Se refiere aquellos datos con que se encuentra siempre el sistema sin antes ser procesador, es decir al inicio de cada sistema, corresponde a los insumos que se requieren para el proceso normal de un sistema

Elementos de salida

Todo sistema debe tener resultado, estos resultados son considerados como elementos de salida, dependiendo del resultado así se determina el desarrollo del sistema, su evolución y para que fueron creado

Procesos

 Teniendo en cuenta los elementos de entrada se ejecutan pasos o funciones que son necesario para la trasformación o modificación de los datos de entrada, los procesos conforman el núcleo de sistema en el están definidos los pasos del sistema
